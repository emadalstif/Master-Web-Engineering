Es wurde Vue.js/VueCLI installiert und eingesetzt, Es ist schwer npm und die zugehörige Tools auf Github hochzuladen, da die aus großen Datenmengen bestehen. Deshalb werden nur die zwei Folder (public und src) der Aufgaben von Vue.js hinzugefügt.

Man kann die Vue.js/ VueClI durch den folgendenen Befehlen installeren
1.Vue Core ---> npm install –save vue
2.Vuex ---> npm install –-save vuex
3.Vue Router ---> npm install –-save vue-router
4.Vue CLI --->npm install -g vue-cli
5.npm run serve
