Es wurde Vue.js/VueCLI installiert und eingesetzt, Es ist schewer npm und die zugehörige Tools schwer auf Github hochzuladen, da die aus großen Datenmengen bestehen.Deshalb werden nur die zwei Folder (public und src) zu den Aufgaben von Vue.js die zwei Folder hinzugefügt.

Man kann die Vue.js/ VueClI durch den folgendenen Befehlen installeren
1.Vue Core ---> npm install –save vue
2.Vuex ---> npm install –-save vuex
3.Vue Router ---> npm install –-save vue-router
4.Vue CLI --->npm install -g vue-cli
5.npm run serve
